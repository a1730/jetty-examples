Example: Embedded Jetty w/ JSP Support 
======================================

This is a maven project, to build it:

    $ mvn clean package

To run the example `examples.Main`:

    $ mvn exec:java

Open your web browser to:

    http://localhost:8080/  

To stop Jetty:

  use <kbd>CTRL</kbd>+<kbd>C</kbd>
