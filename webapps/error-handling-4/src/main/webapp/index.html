Error Handling WebApp

A selection of URL endpoints in this webapp.

URL Expected Result
hello Servlet replies with Hello World
triggers/404 Servlet calls response.sendError(404)
triggers/500 Servlet calls response.sendError(500)
triggers/IOException Servlet throws a java.io.IOException
triggers/Example%20Message%20from%20sendError Servlet calls a response.sendError(500, message)
hello?trigger=404 Filter calls response.sendError(404)
hello?trigger=500 Filter calls response.sendError(500)
hello?trigger=RuntimeException Filter throws a java.lang.RuntimeException
